## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/avs744/Portfolio.git
```

Move into the project folder:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
